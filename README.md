# SegundoCadastrador

1 - Fork/Clone

2 - Install dependencies 
  - `npm install`

3 - Run DB 
  - `psql -U postgres -f consultas.sql`

4 - Run the development server 
  - `npm start`

5 - Test Server
  -  `http://localhost:3000`

6 - Test Queries
  - Listar todos os alunos
  - `http://localhost:3030/api/alunos`
